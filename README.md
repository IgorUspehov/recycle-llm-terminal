# recycle-llm-terminal

**AI / LLM Engineering for Secondary Polymers**

Using Large Language Models via Linux terminal to solve real polymer recycling engineering problems.

## What this is

A portfolio site and technical blog by Ihor Kriazhev — polymer recycling technologist (25+ years, extrusion specialist) who applies LLM tools via CLI to engineering tasks:

- Temperature profile calculation
- Formulation / compound recipe design  
- MFI drift prediction
- Granulator diagnostics
- UV/thermal stabilizer selection

## Site structure

```
index.html          ← main page
tools.html          ← LLM backends and models used
articles/           ← 5 technical articles with real terminal sessions
logs/               ← raw terminal log entries with assessments
contact.html        ← contact and scope of work
css/style.css       ← terminal dark theme
js/terminal.js      ← typing animation
```

## Tools used

- [Ollama](https://ollama.ai) — local LLM backend
- [llm CLI](https://llm.datasette.io) — unified CLI wrapper
- Models: Mistral 7B, CodeLlama 7B, DeepSeek Coder 6.7B
