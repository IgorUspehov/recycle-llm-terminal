Extruder temperature profile via Ollama — HDPE regrind case

Task: calculate a zone-by-zone barrel temperature profile for a single-screw extruder running HDPE regrind (post-consumer canister material, MFI ~0.6 g/10min at 190°C/2.16kg). Output rate target: 250 kg/h. Screw diameter: 90mm. L/D: 30.

The command

ihor@recycle-llm:~$ ollama run mistral
>>> You are a polymer extrusion engineer. Calculate a zone-by-zone barrel temperature profile for a single-screw extruder: material HDPE regrind (post-consumer canisters), MFI 0.6 g/10min (190C/2.16kg), screw D=90mm, L/D=30, target throughput 250 kg/h. Provide: feed zone, compression zone (3 zones), metering zone, adapter, die head. Format as a table with zone name, temperature, and reasoning note.

Model output (Mistral 7B, Q4_K_M)

| Zone | Temp (°C) | Notes |
|-------------------|-----------|----------------------------------------------------|
| Feed zone (Z1) | 170 | Low friction entry, prevent premature melting |
| Compression 1 (Z2)| 200 | Begin melt, HDPE Tm ~130°C, regrind needs +margin |
| Compression 2 (Z3)| 215 | Progressive heating, homogenize melt |
| Compression 3 (Z4)| 220 | Peak compression, avoid degradation ceiling |
| Metering (Z5) | 215 | Slight drop — stabilize melt viscosity |
| Adapter | 210 | Reduce pressure drop, maintain flow |
| Die head | 205 | Final shaping — lower = better surface finish |

WARNING: For regrind material with possible contamination, add +0.2% Irganox 1010
or similar phenolic AO before adjusting metering zone above 220°C.

Recommended screw speed: 75–90 RPM for 250 kg/h at D=90mm.
Estimated melt pressure at die: 180–220 bar.

My assessment

Profile is correct in structure. Feed zone at 170°C is slightly conservative for post-consumer HDPE with possible moisture — I would raise it to 175°C in practice. Die head at 205°C is appropriate for pelletizing; for pipe die you'd want 210–215°C. The Irganox recommendation is relevant and I would actually use it. Screw speed range 75–90 RPM matches what I'd set manually.

What the model can't do: account for specific screw geometry (barrier vs conventional), actual contamination level of the regrind batch, or real-time amperage feedback. It gives a solid starting point — you still need to tune on the machine.

Follow-up prompt for fine-tuning

ihor@recycle-llm:~$ ollama run mistral
>>> Same extruder. If amperage rises 12% above baseline at Z3, what does it indicate and which zone temperatures should I adjust first?
A 12% amperage rise at Z3 (compression zone 2) typically indicates:
1. Insufficient melt temperature — material not fully plasticized before compression
2. Contamination (PP or PS fraction) raising viscosity
3. Screw speed too high for the current feed rate

Adjust first: Raise Z2 (Compression 1) by +5°C, monitor 10 min.
If amperage persists: raise Z3 by +5°C.
Do NOT raise feed zone — overheating feed causes bridging.

This is useful diagnostic logic. Matches my experience. The "do not raise feed zone" note is correct and something a junior operator would miss.

Conclusion

Ollama + Mistral 7B on local hardware can generate a working temperature profile for standard HDPE regrind in under 30 seconds. The output is a reliable starting point that matches practical experience. The model handles follow-up diagnostic questions correctly and does not hallucinate temperatures outside physically reasonable ranges.

Useful for: quick baseline profiles, training junior operators, checking your own reasoning against a second "opinion" before starting a new batch.
