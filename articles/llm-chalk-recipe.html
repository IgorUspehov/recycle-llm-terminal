Chalk filler dosing in HDPE via CLI — formulation experiment

Task: determine the optimal CaCO3 loading in a 70% post-consumer HDPE canister regrind + 30% chalk compound for injection-moulded parts. Requirement: MFI must remain ≥ 4 g/10min (190°C/5kg), notched impact not below 3 kJ/m².

The command

ihor@recycle-llm:~$ llm "formulation: 70% post-consumer HDPE canister regrind MFI 0.8 (190/2.16) + CaCO3 uncoated 3 micron. Target: MFI min 4 g/10min at 190C/5kg after compounding, notched Charpy impact min 3 kJ/m2. What is max CaCO3 % loading, coupling agent needed?"

Model output

Analysis for HDPE regrind MFI 0.8 (190/2.16) + uncoated CaCO3 3µm:

MFI conversion note: MFI 0.8 at 2.16kg ≈ MFI 4.5–5.5 at 5kg for HDPE
(correction factor ~5–7x for HDPE density range 0.955–0.960)

Max CaCO3 loading for MFI target: 25–30%
Above 30%: viscosity rises sharply, MFI at 5kg drops below 3.5 g/10min

Impact warning: uncoated CaCO3 causes stress concentration.
At 30% loading: expected Charpy notched ~2.2–2.8 kJ/m² — BELOW TARGET

Recommendation:
— Use stearic acid coated CaCO3 (0.8–1.2% coating) — impact +35–40%
— OR add 3–5% HDPE-g-MAH coupling agent (e.g. Fusabond E226)
— Combination: 25% coated CaCO3 + 3% compatibilizer = impact ~3.4 kJ/m²

Suggested compound: 72% HDPE regrind + 25% coated CaCO3 + 3% HDPE-g-MAH

My assessment

The MFI correction factor is correct. The impact warning for uncoated CaCO3 is critical and often ignored — I've seen exactly this problem on lines running uncoated filler. The Fusabond recommendation is standard and practical. The 25% coated + 3% compatibilizer formula is a real recipe I would actually run.

Conclusion

The model correctly identified the impact problem with uncoated filler before I mentioned it. That's the useful part — it prompts you to think about what you might have overlooked. Recipe is production-ready as a starting point.
